# Client

## Descricao
Frontend Next.js App Router responsavel pelas telas publicas do quiz, portal admin e portal RH.

## Estrutura
- Rotas em `client/app`
- Componentes de pagina em `client/components`
- Chamadas HTTP e estado de sessao em `client/lib`
- Estilos globais em `client/app/globals.css`

## Dependencias
- `Next.js 15`
- `React 19`
- `socket.io-client`
- `Tailwind CSS 4`
